# 00

## Build

Requirements:

- make
- clang
- SDL2
- Emscripten (Optional -- build for web)

```sh
make
```

## Run

```sh
make run
```

### Optional - Build for Web

```sh
make web
```

Then serve and browse to `00.html`.
